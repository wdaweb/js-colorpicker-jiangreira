
# js-colorpicker-jiangreira
## 隨機RGB色盤



- `+` 將目前RGB三色值都加一
- `-` 將目前RGB三色值都減一
- `?` 隨機產生一組RGB色值
- `#` 色碼可點擊複製

CSS 使用 bootstrap


---
## Rondom ColorPicker

- `+` Press the button and each R,G,B value will plus one
- `-` Press the button and each R,G,B value will subtract one
- `?` Press the button will change random RGB value
- `#` Press text can copy hex

CSS use bootstrap
